# Local Network Recon & Packet Analysis (Nmap + Wireshark)

This project demonstrates practical cybersecurity skills through real network reconnaissance and packet‑level analysis using **Nmap** and **Wireshark**.  
It simulates the workflow of a junior security analyst performing asset discovery, service enumeration, and traffic inspection on a local network.

---

## 🔍 Objectives
- Identify active hosts on a local subnet
- Enumerate open ports and running services
- Capture and analyze live network traffic
- Interpret ICMP, ARP, TCP, QUIC, and mDNS packets
- Document findings in a structured cybersecurity report

---

## 🛠 Tools Used
- **Nmap** (host discovery + service/version detection)
- **Wireshark** (packet capture + protocol analysis)
- **Windows PowerShell**
- **Markdown for documentation**
